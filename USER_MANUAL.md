## Introduction

### What is StayHub?

StayHub is a comprehensive accommodation booking platform that connects guests with hosts, providing a seamless experience for finding and booking accommodations. The platform supports three types of users:

- **Guests**: Travelers looking to book accommodations
- **Hosts**: Property owners who list and manage their properties
- **Admins**: Platform administrators who manage the overall system

### Key Features

- **Property Listings**: Browse and search through various accommodation options
- **Booking Management**: Easy booking process with approval workflow
- **Payment Integration**: Secure PayPal payment processing
- **Messaging System**: Direct communication between guests and hosts
- **Reviews & Ratings**: Rate and review properties and experiences
- **Dashboard Analytics**: Track bookings, earnings, and performance metrics
- **Mobile Responsive**: Access the platform from any device

## Getting Started

### System Requirements

- **Web Browser**: Chrome, Firefox, Safari, or Edge (latest versions)
- **Internet Connection**: Stable internet connection required
- **JavaScript**: Must be enabled in your browser
- **Email Account**: Valid email address for account verification

### Creating an Account

1. **Navigate to Registration**
   - Open your web browser and go to the StayHub website
   - Click on the "Register" or "Sign Up" button

2. **Fill in Registration Form**
   - Enter your email address
   - Create a secure password (minimum 8 characters recommended)
   - Select your user role:
     - **Guest**: If you want to book accommodations
     - **Host**: If you want to list and manage properties
     - **Admin**: If you have administrative access (requires special permissions)

3. **Email Verification**
   - Click "Send Verification Code"
   - Check your email inbox for a 6-digit verification code (OTP)
   - Enter the verification code in the provided field
   - Note: The code expires after 15 minutes

4. **Complete Registration**
   - Click "Verify Email" to complete registration
   - You will be automatically redirected to your dashboard based on your selected role

### Logging In

1. **Access Login Page**
   - Navigate to the StayHub homepage
   - Click "Login" or "Sign In"

2. **Enter Credentials**
   - Enter your registered email address
   - Enter your password
   - Click "Login"

3. **Dashboard Access**
   - After successful login, you'll be redirected to your role-specific dashboard:
     - Guests → Guest Dashboard
     - Hosts → Host Dashboard
     - Admins → Admin Dashboard

### Logging Out

- Click on your profile icon in the top navigation
- Select "Logout" or "Sign Out"
- You will be redirected to the login page

## Guest User Guide

### Guest Dashboard Overview

The Guest Dashboard is your central hub for:
- Browsing available properties
- Managing your bookings
- Viewing your favorites/wishlist
- Accessing your wallet and payment options
- Communicating with hosts
- Managing your profile

### Browsing Properties

#### Search Functionality

1. **Basic Search**
   - Use the search bar at the top of the page
   - Enter location, property name, or keywords
   - Click the search icon or press Enter

2. **Advanced Search Filters**
   - Click on the filter icon to access advanced options
   - Filter by:
     - **Price Range**: Set minimum and maximum price
     - **Date Range**: Select check-in and check-out dates
     - **Number of Guests**: Specify guest count
     - **Amenities**: Filter by available amenities (WiFi, Pool, Parking, etc.)
     - **Property Type**: Filter by category (Apartment, House, Villa, etc.)

3. **Date Selection**
   - Click on the date picker
   - Select your check-in date
   - Select your check-out date
   - Booked dates will be shown as unavailable (grayed out)

#### Viewing Property Details

1. **Property Cards**
   - Browse through property cards showing:
     - Property images
     - Property title and location
     - Price per night
     - Average rating
     - Key amenities

2. **Property Details Page**
   - Click on any property card to view full details
   - View includes:
     - Image gallery (click to view full-size images)
     - Detailed description
     - Complete amenities list
     - Location map
     - Host information
     - Reviews and ratings
     - Availability calendar

3. **Image Gallery**
   - Click on any property image to open the gallery
   - Navigate through images using arrow buttons
   - Close the gallery by clicking the X button or outside the image

### Making a Booking

1. **Select a Property**
   - Browse and find a property you want to book
   - Click on the property to view details

2. **Choose Dates**
   - Select your check-in date
   - Select your check-out date
   - Ensure dates are available (not grayed out)

3. **Select Number of Guests**
   - Use the guest selector to specify the number of guests
   - Ensure the number doesn't exceed the property's maximum capacity

4. **Review Pricing**
   - View the breakdown:
     - Base price per night
     - Number of nights
     - Service fee (platform fee)
     - Total amount

5. **Add Booking Notes (Optional)**
   - Add any special requests or notes for the host
   - This information will be visible to the host

6. **Proceed to Payment**
   - Click "Book Now" or "Reserve"
   - Review booking summary
   - Click "Confirm Booking"

7. **Payment Process**
   - If you have sufficient balance in your PayPal wallet, payment will be processed automatically
   - If not, you'll be redirected to add funds or use PayPal payment
   - Complete the payment process

8. **Booking Confirmation**
   - After successful payment, your booking status will be "Pending Approval"
   - You'll receive a confirmation email
   - Wait for the host to approve or decline your booking

### Managing Bookings

#### Viewing Your Bookings

1. **Access Bookings**
   - Click on "My Bookings" or "Bookings" in the navigation menu
   - View all your bookings organized by status:
     - **Pending Approval**: Waiting for host approval
     - **Upcoming**: Approved bookings with future dates
     - **Completed**: Past bookings
     - **Declined**: Bookings declined by host
     - **Cancelled**: Cancelled bookings

2. **Booking Details**
   - Click on any booking to view:
     - Property details
     - Booking dates
     - Guest count
     - Total amount paid
     - Booking status
     - Host contact information

#### Cancelling a Booking

1. **Select Booking**
   - Go to "My Bookings"
   - Find the booking you want to cancel
   - Click on the booking to view details

2. **Cancel Booking**
   - Click "Cancel Booking" button
   - Confirm the cancellation
   - Note: Cancellation policy applies (typically 25% refund to host, 75% refund to guest)

3. **Cancellation Confirmation**
   - You'll receive a cancellation confirmation email
   - Refund will be processed according to the cancellation policy
   - Booking status will update to "Cancelled"

### Favorites/Wishlist

1. **Adding to Favorites**
   - While browsing properties, click the heart icon on any property card
   - The property will be saved to your favorites

2. **Viewing Favorites**
   - Click on "Wishlist" or "Favorites" in the navigation
   - View all your saved properties
   - Click on any property to view details or make a booking

3. **Removing from Favorites**
   - Click the heart icon again to remove from favorites
   - Or remove from the favorites page

### Wallet & Payments

#### PayPal Wallet

1. **Viewing Balance**
   - Click on "Wallet" in the navigation menu
   - View your current PayPal balance

2. **Adding Funds (Top Up)**
   - Click "Add Funds" or "Top Up"
   - Enter the amount you want to add
   - Choose from preset amounts or enter a custom amount
   - Complete the PayPal payment process
   - Funds will be added to your wallet balance

3. **Transaction History**
   - Click "Transaction History" to view:
     - All payments made
     - Refunds received
     - Top-up transactions
     - Booking payments

### Messaging Hosts

1. **Starting a Conversation**
   - Go to a property listing
   - Click "Message Host" or "Contact Host"
   - Or access messages from your bookings page

2. **Chat Interface**
   - Click on "Messages" in the navigation
   - Select a conversation from the chat list
   - Type your message and press Enter or click Send
   - View message history with the host

3. **Chat Features**
   - Real-time messaging
   - Message notifications
   - Chat history preservation
   - Access from bookings page

### Reviews & Ratings

1. **Leaving a Review**
   - After a completed booking, you can leave a review
   - Go to "My Bookings"
   - Find a completed booking
   - Click "Leave Review" or "Rate Property"

2. **Review Form**
   - Select a star rating (1-5 stars)
   - Write a detailed review about your experience
   - Click "Submit Review"

3. **Viewing Reviews**
   - Reviews are visible on property listing pages
   - View all reviews and ratings from other guests

### Profile Management

1. **Accessing Profile Settings**
   - Click on your profile icon
   - Select "Account Settings" or "Profile"

2. **Edit Profile Information**
   - Update your name
   - Update phone number
   - Change email address (requires verification)
   - Upload profile picture

3. **Change Password**
   - Go to Account Settings
   - Click "Change Password"
   - Enter current password
   - Enter new password
   - Confirm new password
   - Click "Update Password"

4. **Notification Preferences**
   - Manage email notifications
   - Manage SMS notifications
   - Set booking update preferences

### Sharing Properties

1. **Share Property Listing**
   - Click the share icon on any property
   - Choose sharing method:
     - Copy link
     - Share via social media (Facebook, Twitter, etc.)
     - Share via messaging apps (WhatsApp, Telegram, etc.)

2. **Share Booking**
   - From your bookings page, click share on a booking
   - Share booking details with others

- **Platinum Host**: 18-23 bookings

- **Master Host**: 30-35 bookings
- **Elite Host**: 36-41 bookings
- **Legend Host**: 42+ bookings

- **Points System**: Earn 50 points per booking
- **Benefits**: Higher levels may unlock special features and benefits

## Troubleshooting

### Login Issues

**Problem**: Cannot log in
- **Solution**: 
  - Verify email and password are correct
  - Check if Caps Lock is on
  - Try resetting password
  - Clear browser cache and cookies
  - Try a different browser

**Problem**: Forgot password
- **Solution**: 
  - Click "Forgot Password" on login page
  - Enter your email
  - Check email for password reset link
  - Follow instructions to reset password

### Booking Issues

**Problem**: Cannot select dates
- **Solution**: 
  - Dates may be already booked
  - Check-in date must be in the future
  - Check-out date must be after check-in
  - Clear browser cache and try again

**Problem**: Payment failed
- **Solution**: 
  - Check PayPal account balance
  - Verify payment method is valid
  - Check internet connection
  - Try again after a few minutes
  - Contact support if issue persists

**Problem**: Booking not showing
- **Solution**: 
  - Refresh the page
  - Check booking status
  - Verify you're logged into the correct account
  - Check "My Bookings" section

### Host Issues

**Problem**: Listing not appearing
- **Solution**: 
  - Check if listing is published (not inactive)
  - Verify all required fields are filled
  - Check if images are uploaded
  - Refresh the page

**Problem**: Not receiving booking requests
- **Solution**: 
  - Check listing visibility status
  - Verify pricing is competitive
  - Ensure availability calendar is updated
  - Check notification settings

### Payment Issues

**Problem**: Wallet balance not updating
- **Solution**: 
  - Refresh the page
  - Check transaction history
  - Wait a few minutes for sync
  - Contact support if balance doesn't update

**Problem**: Refund not received
- **Solution**: 
  - Check transaction history
  - Verify cancellation was processed
  - Allow 3-5 business days for refund
  - Contact support with booking ID

### Technical Issues

**Problem**: Page not loading
- **Solution**: 
  - Check internet connection
  - Clear browser cache
  - Try a different browser
  - Disable browser extensions
  - Check if JavaScript is enabled

**Problem**: Images not displaying
- **Solution**: 
  - Check internet connection
  - Refresh the page
  - Clear browser cache
  - Try a different browser

**Problem**: Messages not sending
- **Solution**: 
  - Check internet connection
  - Refresh the page
  - Try sending again
  - Check if recipient is online

### General Issues

**Problem**: Email verification not received
- **Solution**: 
  - Check spam/junk folder
  - Verify email address is correct
  - Wait a few minutes
  - Request new verification code
  - Check email service status

**Problem**: Profile not updating
- **Solution**: 
  - Ensure all required fields are filled
  - Check for validation errors
  - Save changes after editing
  - Refresh the page
